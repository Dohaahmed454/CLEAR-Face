# Privacy, Retention, and Deletion Plan 
 
## Privacy 
 - Use anonymous participant IDs. 
 - Store real face data outside the public GitHub repository. 
 - Do not commit signed consent forms to GitHub. 
 - Limit access to authorized researchers only. 
 
## Retention 
 - Retain data only for the approved research period. 
 - Store raw real-face data separately from code and documentation. 
 
## Deletion 
 - Maintain a participant-ID deletion map in a private secure location. 
 - Delete participant data upon valid withdrawal request where permitted by the ethics protocol. 
 - Record deletion completion in a private audit log.
