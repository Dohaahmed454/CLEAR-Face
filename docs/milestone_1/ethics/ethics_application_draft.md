# Ethics Application Draft 
 
## Study purpose 
 
This study investigates whether a facial-expression recognition system can request a better visual observation when the current observation is unreliable. 
 
## Data collected 
 - Face images or short face video sequences. 
 - System prompts and participant responses. 
 - Timing information. 
 - Compliance with requested actions. 
 - Technical quality measurements. 
 
## Data not collected or inferred 
 
The study does not perform identity recognition, age prediction, gender classification, personality inference, deception detection, mental-health inference, or claims about true internal emotional state. 
 
## Risks 
 - Privacy risk from facial data. 
 - Participant discomfort with being recorded. 
 - Misinterpretation of expression labels. 
 
## Mitigation 
 - Use anonymized participant IDs. 
 - Avoid collecting unnecessary personal information. 
 - Store data securely. 
 - Allow withdrawal and deletion. 
 - Avoid prohibited internal-state claims. 
 - Report results using responsible terminology.
