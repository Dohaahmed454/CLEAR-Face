## Main FER dataset

Candidate:
- Dataset name: AffectNet
- Expression labels: Happy, Sad, Surprise, Fear, Disgust, Anger, Contempt, Neutral
- Number of images/videos: ~1M images
- Participant IDs available? No
- License: Research use (non-commercial)
- Download source: https://mohammadmahoor.com/affectnet/
- Reason for selection: Large-scale in-the-wild dataset suitable for real-world emotion recognition under noise, occlusion, and lighting variation.




## External FER dataset

Candidate:
- Dataset name: RAF-DB
- Expression labels: Happy, Sad, Surprise, Fear, Disgust, Anger, Neutral
- Number of images/videos: ~30,000 images
- Participant IDs available? No
- License: Research use
- Download source: http://www.whdeng.cn/RAF/model1.html
- Reason for selection: Standard benchmark used for testing generalization of FER models on unseen data distributions.


## Selection criteria 
 
The selected datasets should support: 
 
1. Facial-expression recognition of visible facial 
configurations. 
2. Clear label mapping to the project expression classes. 
3. Participant-disjoint splits where possible. 
4. Controlled degradation experiments. 
5. External testing to measure generalization. 
6. Ethical and licensing compatibility.
