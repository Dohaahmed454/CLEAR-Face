Experimental Protocol
Failure conditions

The project is limited to four main failure conditions:

1-Poor lighting.
2-Blur or motion.
3-Non-frontal pose.
4-Small or distant face.
5-CARE decisions

CARE is limited to six decisions:

1-Predict.
2-Request improved lighting.
3-Request facing the camera.
4-Request moving closer.
5-Request holding still or capturing a short sequence.
6-Abstain.
7-Scope rule

No additional failure condition or CARE action should be added unless it directly strengthens the CARE -> CLEAR-Face-R -> explanation chain.
